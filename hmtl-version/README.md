# Versao HTML/CSS/JS (Inicial)

Esta pasta contem a versao mais simples do projeto, feita apenas com:

- HTML
- CSS
- JavaScript

Objetivo: dar o primeiro contato com desenvolvimento web sem frameworks.

## O que os alunos podem aprender aqui

- Estrutura basica de uma pagina HTML
- Estilizacao com CSS puro
- Manipulacao do DOM com JavaScript
- Eventos de interface (`change` em selects)
- Renderizacao dinamica de tabela com dados locais

## Estrutura de arquivos

- `index.html`: estrutura da pagina e elementos principais
- `styles.css`: estilos da interface
- `script.js`: dados de exemplo, filtro, ordenacao e renderizacao

## Como executar

Opcao 1: abrir diretamente no navegador

- Abra `hmtl-version/index.html` no browser.

Opcao 2: rodar servidor local (recomendado para aula)

```bash
python3 -m http.server 5500
```

Depois acesse:

`http://localhost:5500/hmtl-version/`
