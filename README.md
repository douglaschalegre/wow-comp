# Repo de ensino (3 versoes)

Este repositorio contem 3 implementacoes do mesmo site (complexidade crescente), uma por pasta:

- `hmtl-version/`: HTML/CSS/JS puro (DOM + eventos), sem build
- `react-version/`: mesma UI em React (componentes/estado), com Vite
- `next/`: Next.js (app fullstack: rotas/API, env, DB/Prisma/jobs)

Para rodar, veja o `README.md` dentro de cada pasta.
